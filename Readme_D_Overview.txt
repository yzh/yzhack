# Dungeon Map Overview 3 [Hojita Discordia]
# based patch -- 2006.12.02


* Dungeon Map Overview 日本語版 [Shiganai Sakusha]

JNetHack3.4.3-0.10用のパッチです。
以下の機能を追加します。

    キー操作      #コマンド   機能
 ^O(Metaキー+o)  #overview 迷宮の大まかな構造を表示する。
                           到達済みの階層や発見済みの分岐・一部施設を大まかに表示します。
  
 ^N(Metaキー+n)  #annotate  現在いる階に名前をつける。
                            つけた階層名は^Oでの一覧に表示されます。


また日本語版では以下の機能を追加しています。
-wizardモード時の^Oは、2種類の表示を選択できます。
--簡易表示:通常プレイ時の^O表示と同様。到達済みの階層や発見済みの分岐・一部施設を大まかに表示します。
--詳細表示:従来のwizardモード時の^O表示と同様。迷宮全体のステータス(具体的な特殊階の位置や分岐など)を全て表示します。
